# Stock Market Prediction

## Overview
This project predicts stock prices using machine learning models.

## Models Used
- Linear Regression
- Random Forest
- XGBoost

## Metrics
- MSE
- RMSE
- MAE

## How to Run
1. Install requirements
2. Delete the CSV in raw
3. Run src/data_loader.py
4. Run the exploratory_data_analysis notebook to check insights from the data
5. Run main.py
